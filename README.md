##### Install SCRIPT !!!

`````
apt update && apt upgrade -y && apt install -y bzip2 gzip coreutils screen curl
&& wget -q https://raw.githubusercontent.com/ibanstore/script/main/iban.sh && chmod
+x iban.sh && ./iban.sh
`````

